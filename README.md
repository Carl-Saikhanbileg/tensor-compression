# Tensor Network Structure Search (Prototype)

Implementation of a prototype inspired by Guo et al. (2026):

- tensor_core.py: tensor + matricization
- preprocessing.py: singular value preprocessing
- sketch.py: OSplit sketches
- scoring.py: rank assignment (greedy)
- execution.py: actual decomposition
- search.py: orchestration

Work in progress toward full TN-SS reproduction.
